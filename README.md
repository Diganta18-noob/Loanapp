## 🌟 About The Project

**LoanApp** is an innovative financial application that leverages cutting-edge AI (Google Gemini & Groq SDK) to provide intelligent loan prediction, processing, and an interactive intelligent Chatbot assistant.

Featuring a stunning visual aesthetic powered by Material UI and Framer Motion, it offers users an immersive, premium, and trustworthy financial experience.

### ✨ Key Features

- 🤖 **AI Chatbot Assistant**: Specialized in answering financial queries and loan estimations leveraging Gemini/Groq.
- 💳 **Loan Processing Engine**: Submit, analyze, and process loan applications with real-time feedback.
- 🎨 **Dynamic Interface**: Cinematic UI interactions and fluid animations ensuring a high-end user journey.
- 🔐 **Secure & Scalable**: Solid Express/Mongoose backend architecture equipped with JWT authentication and Bcrypt.
- 📱 **Platform Ready**: Fully responsive layout designed perfectly for desktops, tablets, and smartphones.

---

## 💻 Tech Stack

### Frontend (Client & Chatbot)

- **Framework**: React 19.x built with Vite
- **Global State**: Redux Toolkit & React-Redux
- **Styling & UI**: `@mui/material`, Emotion, `react-icons`
- **Animations**: Framer Motion
- **Tooling**: Axios, React Toastify

### Backend (Server)

- **Framework**: Node.js & Express
- **Database**: MongoDB (Mongoose ORM)
- **AI Integrations**: `@google/generative-ai`, `groq-sdk`
- **Utilities**: Multer (File uploads), JWT, Bcryptjs

## 🚀 Getting Started

Quickly boot up the LoanApp environment locally.

### Prerequisites

- Node.js (v18+)
- MongoDB instance (Local or remote)
- API Keys for Google Gemini / Groq

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/Diganta18-noob/Loanapp.git
   cd Loanapp
   ```

2. **Server Setup**

   ```bash
   cd server
   npm install
   # Create a .env file with MONGO_URI, JWT_SECRET, and AI API keys
   npm run dev
   ```

3. **Client Setup**

   ```bash
   cd ../client
   npm install
   npm run dev
   ```

4. **Experience the App!**
   Navigate to the local Vite URL outputted in your terminal and let the AI guide you.
